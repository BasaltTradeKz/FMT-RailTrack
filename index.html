CallMeBot — бесплатный способ отправки в WhatsApp
Каждый получатель должен активировать бота один раз — написать в WhatsApp боту и получить API-ключ.
1
Добавьте номер +34 644 59 67 47 в контакты (CallMeBot)
2
Напишите этому контакту: I allow callmebot to send me messages
3
Через минуту получите API-ключ в ответном сообщении
4
Добавьте получателей на вкладке «Получатели»
Twilio WhatsApp API — надёжный коммерческий провайдер.
Регистрация: twilio.com · Sandbox для тестирования бесплатный.
Формат: whatsapp:+14155238886 (Twilio Sandbox) или ваш одобренный номер
⚠️ Twilio-запросы идут через backend-сервер. Для работы прямо из браузера используйте Python-скрипт whatsapp_sender.py
WhatsApp Business API (Meta) — официальный корпоративный канал.
Требует верификацию бизнеса в Meta. Подходит при объёмах 100+ сообщений/день.
⚠️ WABA также требует backend. Используйте whatsapp_sender.py для интеграции.
Номер в формате +7XXXXXXXXXX (Казахстан) или +998XXXXXXXXX (Узбекистан)
Для CallMeBot: у каждого получателя должен быть свой API-ключ после активации бота
API-ключи CallMeBot (по одному на получателя)
Переменные: {date} {time} {total} {loaded} {empty} {max_idle} {avg_idle} {stations} {top3}
Переменные: {station} {count} {max_idle} {wagons_list}
